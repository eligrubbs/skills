# skills
My Personal Repository of Skills for AI Agents

This is 99% copied from matt pocock's [skills repository](https://github.com/mattpocock/skills/tree/main) with tweaks for my workflows.

## Installation

These are just text files that follow the a agent skills [specification](https://agentskills.io/specification).

I use Pi agent primarily, so there will eventually be a short guide / script on how to get that configured.

## Motivation

AI seems very powerful. I hear stories of people harnessing them to move very quickly while still producing high quality software.

I want to invest time understanding the myths, and siphoning out the pieces that work and bring value.

I always want to remain in-the-loop with the state of the software I am writing.
